Vogel法：

伏格尔法的步骤是从运价表上分别找出每行与每列的最小的两个元素之差，再从差值最大的行或列中找出最小运价确定供需关系和供应数量。当产地或销地中有一方数量上供应完毕或得到满足时，划去运价表中对应的行或列，再重复上述步骤。在产销地数量不多时，Vogel法给出的初始方案一般就是最优方案，所以Vogel法有时就用作求运输问题最优方案的近似解。
